👤 Profile Card Component

A responsive Profile Card Component built as part of a Frontend Mentor challenge.
The project focuses on creating a modern profile UI with user information and statistics using HTML and CSS.

🚀 Features

- Profile card layout
- Background pattern design
- Profile avatar image
- User information (name, age, location)
- Statistics section (followers, likes, photos)
- Responsive centered layout
- Clean modern UI design

| Technology       | Purpose                 |
| ---------------- | ----------------------- |
| **HTML5**        | Semantic page structure |
| **CSS3**         | Styling and layout      |
| **Flexbox**      | Layout alignment        |
| **GitHub Pages** | Deployment              |
